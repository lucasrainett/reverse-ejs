reverse-ejs

Turn any web page into structured JSON. Define an EJS template over the HTML and extract the data instantly.

GitHub npm Docs

How it works

1

Save the HTML

View source on any page and copy the HTML you want to extract data from.

2

Create a template

Replace the values you care about with EJS tags like <%= name %>.

3

Extract the data

Paste both below and get a clean JSON object with every value extracted.

Features

  • Loops & nested data - extract arrays and nested objects from repeating HTML sections
  • Conditionals - if/else, switch/case, and boolean detection (including complex conditions like a > b)
  • JS expression capture - tags like <%= price * qty %> or <%= name.toUpperCase() %> are captured under the raw expression text as the key
  • Flexible whitespace - matches pretty-printed and minified HTML against the same template
  • Type coercion - convert extracted strings to numbers, booleans, or dates via the types option
  • Compiled templates - compileTemplate() and reverseEjsAll() for fast batch extraction
  • Safe mode - return null on no-match instead of throwing
  • HTML entity unescaping - &amp;, &lt;, &quot; automatically decoded
  • Includes / partials - reuse template fragments across pages
  • Full TypeScript types - hover docs in your editor for every option
  • Zero dependencies - 20KB, runs in any browser, Node.js 20+, Bun, or Deno
  • Privacy-first - the playground runs entirely in your browser, your HTML never leaves your device
