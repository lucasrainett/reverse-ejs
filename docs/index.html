reverse-ejs

Turn any templated text into structured JSON. Works with HTML, Markdown, logs, emails, plain text - anything you can describe with an EJS template.

Read the Article GitHub npm Docs

How it works

1

Grab the text

Copy the HTML page, Markdown document, log line, email, or any other templated text you want to extract data from.

2

Create a template

Replace the values you care about with EJS tags like <%= name %>.

3

Extract the data

Paste both below and get a clean JSON object with every value extracted.

Playground

Paste your rendered text and EJS template, then hit Extract.

Features

				
    
					
  • 
						Works with any text format - HTML,
						Markdown, logs, emails, CSV, plain text - anything you
						can describe with an EJS template
					
    • 
					
  • 
						Loops & nested data - extract arrays
						and nested objects from any repeating section
					
    • 
					
  • 
						Conditionals - if/else, switch/case,
						and boolean detection
					
    • 
					
  • 
						Flexible whitespace - matches
						pretty-printed and minified HTML against the same
						template
					
    • 
					
  • 
						Type coercion - convert extracted
						strings to numbers, booleans, or dates
					
    • 
					
  • 
						Includes / partials - reuse template
						fragments across pages
					
    • 
					
  • 
						Zero dependencies - 20KB, runs in any
						browser, Node.js, Bun, or Deno
					
    • 
					
  • 
						Privacy-first - everything runs in your
						browser, your data never leaves your device
